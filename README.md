# 🏡 Airbnb Clone – Full Stack MERN Application

A feature-rich Airbnb-inspired web application built with the **MERN Stack**, enabling users to discover, list, and reserve properties through a modern, responsive interface. The project demonstrates full-stack development concepts including authentication, REST APIs, database management, and real-world CRUD operations.

---

## ✨ Overview

This project replicates the core functionality of Airbnb by allowing users to:

* Create an account and securely authenticate
* Explore available properties
* List their own properties
* Save favorite listings
* Book accommodations
* Manage reservations and listings

The application follows a scalable architecture with a React frontend, Express/Node.js backend, and MongoDB database.

# ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/airbnb-clone.git
```

### Navigate to Project

```bash
cd airbnb-clone
```

### Install Dependencies

Backend

```bash
cd server
npm install
```

Frontend

```bash
cd client
npm install
```

---

# 🔐 Environment Variables

Create a `.env` file inside the server folder.

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```

---

# ▶ Running the Application

### Backend

```bash
cd server
npm run dev
```

### Frontend

```bash
cd client
npm start
```

The application will be available at:

```
Frontend : http://localhost:3000

Backend : http://localhost:5000
```
